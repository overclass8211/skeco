## 2. 페이지별 화면 명세

### 2.1 로그인 (`/login`)

| 영역 | 구성 |
|------|------|
| 헤더 | 로고 + 시스템 명 |
| 본문 | ID, 비밀번호, 로그인 버튼, OTP 입력 (2단계 시) |
| 하단 | "Google 로그인" (OAuth 2.0) |

**입력 필드**:
| 필드 | 타입 | 검증 |
|------|------|------|
| username | text | 필수, 3~50자 |
| password | password | 필수, 8자+ |
| otp | number | 6자리 (OTP 활성 시) |

### 2.2 대시보드 (`#dashboard`)

```
┌──────────────────────────────────────────────────┐
│ 📊 대시보드                          [연도 필터▼] │
├──────────────────────────────────────────────────┤
│ [KPI 1] [KPI 2] [KPI 3] [KPI 4] [KPI 5]          │
├─────────────────────┬────────────────────────────┤
│ 월별 신규 리드 차트  │ 파이프라인 펀넬             │
├─────────────────────┴────────────────────────────┤
│ 지역별(국내/해외) 현황                            │
├──────────────────────────────────────────────────┤
│ 최근 활동 타임라인 (10건)                         │
└──────────────────────────────────────────────────┘
```

**5대 KPI**:
1. 활성 리드 수
2. 입찰 진행 건수
3. 올해 수주액 (억원)
4. 수주율 (%)
5. 평균 단가 (백만원)

- 마감일 D-N

## 3. 공통 UI 컴포넌트

### 3.1 모달 (Modal)
- 중앙 정렬
- 어두운 오버레이 (rgba(0,0,0,.45))
- ESC로 닫기
- 다크모드 자동 대응

### 3.2 토스트 (Toast)
- 우측 하단 출현
- 4초 자동 닫힘
- 색상: success(녹색) / warn(주황) / error(빨강)

### 3.3 배지 (Badge)
- 단계: `badge-stage-{lead/review/...}`
- 색상: `badge-{red/green/blue/gray}`

### 3.4 칸반 카드
- 드래그 핸들 (전체 영역)
- hover 효과 (border-color)

## 4. 모바일 / PWA 화면

| 영역 | 변경 |
|------|------|
| 사이드바 | 햄버거 메뉴 (768px 이하) |
| 입력 폰트 | 16px 강제 (iOS auto-zoom 방지) |
| 페이지 타이틀 | white-space: nowrap (세로 깨짐 방지) |
| 캘린더 | 모바일 뷰 자동 전환 |

## 5. 다크모드 가이드

모든 흰색 영역은 `var(--surface)` 변수 사용 — 다크모드 자동 전환:
- `--surface`: #FFFFFF (라이트) / #25232a (다크)
- `--text-1`: #1F2329 / #ece8e6
- `--border`: #E4E7EB / #3d3a44
